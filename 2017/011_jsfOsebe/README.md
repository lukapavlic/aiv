# JSF - naprednejši primer

Primer demonstrira pravilno uporabo upravljanih zrn (session scoped in request scoped).
Prikazuje tudi razliko med actionListener in action atributi - actionlistener proži metodo, action pa po izvedbi metode poskrbi za preusmeritev
Prikazani so tudi validatorji in konverterji (datum pri prikazu)
Bodite pozorni na to, kako je izvedena izbira vrstice v tabeli in preusmerite na podrobnosti - privzeto v JSF vse poteka s pomoèjo POST zahtev
