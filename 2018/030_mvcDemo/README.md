# MVC

Zelo preprosto ogrodje z vzorcem MVC

Krmilnik skrbi za delegiranje opravil poslovnim razredom (implementirajo vmesnik IAkcija)

Po izvedeni kodi (IAkcija.izvediAkcijo) krmilnik prikaz podatkov preusmeri na doloèeno stran JSP

Komunikacija mad poslovno logiko in prikazom poteka preko atributov HTTP zahteve (poslovna logika dopolni isto zahtevo, kot jo kasneje prejme prikaz).
